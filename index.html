Conceito de JAVA

Para que serve?

Java é uma linguagem de programação extremamente versátil. Pode ser usado para programar softwares e aplicativos para muitos sistemas operacionais e dispositivos diferentes, incluindo dispositivos IoT, celulares e computadores

Vantagens

Orientado a objetos: O Java permite criar programas modulares e reutilizar código.

Segurança: O Java foi concebido para ser seguro.

Fácil de aprender: O Java foi projetado para ser fácil de usar, escrever, compilar e depurar.

Desvantagens

Consumo de memória: O Java pode consumir muita memória.

Lentidão na inicialização: O Java pode ser lento para inicializar.

Sobrecarga de tempo de execução: O Java pode ter sobrecarga de tempo de execução.
